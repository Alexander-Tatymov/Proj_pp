# Отчёт ЛР8 — TaskTracker
## Что сделано
- Создан TaskValidator с правилами:
- Title обязателен, максимум 50 символов
- Description максимум 300 символов
- Валидация подключена в:
- Add
- Update
- Import (перед заменой задач)
## Как проверить
1) Добавить пустой Title -> ошибка
2) Добавить Title > 50 -> ошибка
3) Редактировать Description > 300 -> ошибка
4) Импортировать JSON с пустым Title -> импорт отменён
