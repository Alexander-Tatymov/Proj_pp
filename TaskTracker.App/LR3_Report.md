# Отчёт ЛР3 — TaskTracker
## Что сделано
- В TaskService добавлены методы:
- ChangeStatus(id, status)
- Delete(id)
- В консольном меню добавлены пункты:
- 3) Изменить статус задачи
- 4) Удалить задачу
- Добавлена проверка ввода Id и статуса (TryParse и диапазон)
## Как проверить
1) Запуск (F5)
2) Добавить 2 задачи
3) Изменить статус задачи Id=1 на Done
4) Удалить задачу Id=2 (подтверждение y)
5) Проверить список
6) Проверить ошибки:
- Id = abc
- Id = 999
- Status = 5
