# Отчёт ЛР7 — TaskTracker
## Что сделано
- Добавлены функции:
- Backup tasks.json в папку backups
- Export задач в отдельный файл в папке exports
- Import задач из выбранного JSON-файла (замена списка)
- Добавлен метод ReplaceAll в TaskService для импорта
- Добавлена обработка ошибок (файл не найден, неверный путь)
## Как проверить
1) Запустить программу (F5)
2) Добавить 2 задачи
3) Нажать 9 -> создать backup
4) Нажать 10 -> экспорт, проверить файл в exports
5) Нажать 11 -> импорт из экспортированного файла, подтвердит
ь y
6) Показать список (2) -> задачи восстановились/заменились
